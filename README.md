Install
-------
open config/database.php and configure the next variables :<br>
`$DB_NAME = 'camagru';`<br>
`$DB_USER =  'root';`<br>
`$DB_PASSWORD = 'root';`<br>

then go to `/config/setup.php` to install the database.
